Fourth milestone: We Have a Firewall!

Wed, Feb 8, 2017

Igalian Asumu Takikawa has completed the next development milestone of SnabbWall. This means that the L7fw application is now available, as well as a snabb wall filter command, which can be readily used. With these additions, the fourth development milestone of our development roadmap is now completed. Make sure to check out Asumu’s complete writeup about the new features.

We have also started using the project wiki to write down some technical information for developers wanting to use the SnabbWall components in their Snabb applications, or looking into contributing to the project.
